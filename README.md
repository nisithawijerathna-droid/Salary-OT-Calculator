# Salary & OT Calculator

A clean, modern, and accurate **Salary and Overtime Calculator** built for Sri Lankan payroll needs.

### Features:
- Calculates increment based on **Current Gross Salary**
- Supports **Officer Level (240 hours)** and **Assistant Level (200 hours)**
- Accurate Normal OT (1.5x) and Double OT (2x) calculations
- 8% deduction + fixed **Welfare deduction of 99.00**
- Beautiful UI with smooth animations
- Fully responsive and user-friendly

### How it works:
- Enter Current Basic Salary, Current Gross Salary, and Increment %
- Select employee level (Officer / Assistant)
- Input Normal OT and Double OT hours
- Get instant breakdown: New Basic, OT rates & amounts, Gross Salary, Deductions, and **Net Salary**

### Technologies:
- HTML5
- CSS3 (with modern animations)
- Vanilla JavaScript
